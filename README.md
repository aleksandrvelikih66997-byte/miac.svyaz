
# МИАЦ.СВЯЗЬ (МИАЦ.АТС)

Профессиональный веб-интерфейс управления IP-АТС Asterisk 20, оптимизированный для работы в среде **AltLinux SP 10**.

## 🛠 Статус интеграции
Система работает в гибридном режиме для обеспечения максимальной безопасности (ФСТЭК):
1. **Frontend/DB**: Управление абонентами и транками через Firestore.
2. **Backend (Asterisk)**: Локальный сервер Asterisk на AltLinux.
3. **Синхронизация**: Через экспорт `.conf` файлов.

## 🚀 Как запустить абонентов

Если вы видите ошибки `Request 'REGISTER' failed` в логах Asterisk:

### 1. Подготовка файлов
1. В веб-интерфейсе перейдите в раздел **Управление**.
2. Нажмите **PJSIP Абоненты** — скачается файл `pjsip_miac_users.conf`.
3. Загрузите его на сервер в папку `/etc/asterisk/`.

### 2. Настройка прав (важно!)
Если Asterisk запущен, но не дает доступа к консоли:
```bash
# Проверить статус
systemctl status asterisk

# Установить владельца на сокет и конфиги
chown -R asterisk:asterisk /var/run/asterisk
chown -R asterisk:asterisk /etc/asterisk
chmod 770 /var/run/asterisk/asterisk.ctl
```

### 3. Применение настроек
```bash
# Перезагрузить конфигурацию без остановки звонков
asterisk -rx "core reload"

# Проверить, появились ли абоненты
asterisk -rx "pjsip show endpoints"
```

## 📝 Структура PJSIP (AltLinux)
Проект использует транспорт `transport-udp` (порт 5060). Все абоненты подключаются с использованием `auth_type=userpass`.

## 🛡 Безопасность
Все секреты (пароли SIP) хранятся в зашифрованном виде в базе данных и передаются на сервер только через защищенный экспорт конфигурации.
