# **App Name**: АльтернаТИВ АТС

## Core Features:

- Текущий Статус АТС: Отображение ключевых метрик и состояния сервера Asterisk (активные звонки, статус каналов, информация о загрузке) через AMI-запросы.
- Управление Экстеншенами: Добавление, изменение и удаление внутренних номеров (экстеншенов) и привязанных к ним настроек SIP/PJSIP напрямую через AMI-запросы к Asterisk.
- Настройка Транков: Конфигурирование SIP-транков для входящих и исходящих вызовов, включая параметры аутентификации и регистрации.
- Маршрутизация Вызовов: Управление входящими и исходящими маршрутами (диалпланом), позволяющее определить логику обработки вызовов.
- Менеджер Служб Asterisk: Функционал для безопасного запуска, перезапуска и остановки служб Asterisk, а также просмотра консольного вывода.
- Интеллектуальный Помощник по Конфигурации: AI-tool для предложения оптимальных конфигураций экстеншенов или маршрутов на основе заданных пользователем общих параметров (например, 'Создать группу для отдела продаж с очередью').

## Style Guidelines:

- Схема: Светлая. Фоновый цвет: Чистый и спокойный серый (#ECF0F4), напоминающий цвет чистого рабочего пространства.
- Основной цвет: Профессиональный синий (#2B6EBE) для акцента на надежности и функциональности, обеспечивает хороший контраст на светлом фоне.
- Акцентный цвет: Свежий циан (#1CCDE5) для интерактивных элементов, привлекающий внимание к действиям.
- Заголовки и основной текст: 'Inter' (sans-serif) для современного, машинного, объективного и нейтрального вида, обеспечивая отличную читаемость для технических интерфейсов.
- Использовать простые, чистые контурные иконки, ассоциирующиеся с телефонией (например, телефонная трубка, шестерёнка, пользователи, маршруты).
- Предпочтительна приборная панель (dashboard) с боковой навигационной панелью для быстрого доступа к различным функциям, с центральной областью для отображения и управления содержимым. Макет должен быть отзывчивым.
- Использовать минималистичные переходы для уведомлений, загрузки данных и при взаимодействии с элементами форм для улучшения пользовательского опыта, но без отвлечений.
