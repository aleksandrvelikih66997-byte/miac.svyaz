
# Руководство по обеспечению безопасности (ФСТЭК Ready)
## МИАЦ.СВЯЗЬ — Панель управления Asterisk v1.0

Система разработана для применения в критических информационных инфраструктурах и изолированных сегментах сети на базе **AltLinux SP 10**.

### 1. Архитектура и Изоляция (Air-Gap)
- **Zero Internet Dependency**: Полностью удалены зависимости от внешних CDN. Шрифты Google Fonts заменены системным стеком ОС. Иконки поставляются в составе локального пакета.
- **Local Bridge**: Взаимодействие с Asterisk происходит через локальный интерфейс AMI (127.0.0.1).
- **Static Assets**: Все скрипты и стили хранятся на локальном сервере.

### 2. Идентификация и Учет (Аудит)
- **Расширенное логирование**: Каждое действие администратора фиксируется в файле `src/data/audit_logs.json`.
- **Состав лога**: Время, идентификатор администратора, тип действия (вход, создание номера, удаление транка) и детали объекта.
- **Хеширование паролей**: Используется алгоритм SHA-256 с солью для исключения хранения паролей в открытом виде.

### 3. Защита сессий и передачи данных
- **Session Security**: Куки сессии используют флаг `HttpOnly`, что предотвращает доступ к ним через JS (защита от XSS).
- **Timeouts**: Время жизни сессии ограничено 8 часами или завершается при закрытии браузера.
- **RBAC**: Реализовано четкое разделение ролей (Администратор/Оператор).

### 4. Целостность и Конфиденциальность
- **Validation**: Все входящие данные из веб-интерфейса проходят строгую валидацию перед генерацией файлов конфигурации Asterisk.
- **Access Control**: Доступ к файлам конфигурации в `/etc/asterisk/` регулируется на уровне прав доступа файловой системы ОС AltLinux.
- **Secret Management**: Пароли SIP хранятся в защищенных локальных JSON и генерируются в `.conf` файлы только по мере необходимости.
